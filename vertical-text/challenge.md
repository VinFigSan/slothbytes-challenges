# Vertical Text
Create a function that converts a string into a matrix of characters that can be read vertically. Add spaces when characters are missing.  

## Examples
> vertical_txt("Holy bananas")
> output = [
  ["H", "b"],
  ["o", "a"],
  ["l", "n"],
  ["y", "a"],
  [" ", "n"],
  [" ", "a"],
  [" ", "s"]
]
> 
> vertical_txt("Hello fellas")  
> output= [
  ["H", "f"],
  ["e", "e"],
  ["l", "l"],
  ["l", "l"],
  ["o", "a"],
  [" ", "s"]
]
